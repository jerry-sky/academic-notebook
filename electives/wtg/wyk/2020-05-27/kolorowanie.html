Def Kolorowanie grafu

Mamy G=(V,E)G = (V,E)G=(V,E).

Kolorowaniem grafu nazywamy funkcję f:V→N+f: V \to \mathbb{N}^+f:V→N+

Def Właściwe kolorowanie grafu

Mamy G=(V,E)G = (V,E)G=(V,E).

Właściwym kolorowaniem grafu nazywamy funkcję f:V→N+f: V \to \mathbb{N}^+f:V→N+ taką, że (∀x,y∈V)({x,y}∈E→c(x)≠c(y))(\forall x,y \in V)(\{x,y\} \in E \to c(x) \neq c(y))(∀x,y∈V)({x,y}∈E→c(x)=c(y))

Uwaga: będziemy się zajmować tylko grafami prostymi (wielokrotność krawędzi nie ma znaczenia; graf nie może mieć pętli).

Input: (c;x1,x2,…,xk)(c; x_1,x_2, \dots, x_k)(c;x1​,x2​,…,xk​)

    1. f:=min⁡(N+∖{c(y):{x,y}∈E∧y∈domain(c)})f := \min ( \mathbb{N}^+ \setminus \{ c(y): \{x,y\} \in E \land y \in \mathrm{domain}(c) \} )f:=min(N+∖{c(y):{x,y}∈E∧y∈domain(c)})
    2. c:=c∪{(xi,f)}c := c \cup \{ (x_i,f) \}c:=c∪{(xi​,f)}
  2. return ccc

Def Graf kkk-zdegenerowany
