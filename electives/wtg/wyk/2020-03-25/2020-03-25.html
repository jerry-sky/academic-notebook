TwierdzenieTwierdzenieTwierdzenie (Cayley)

Niech n≥2n \ge 2n≥2.

Dowód TwierdzeniaTwierdzeniaTwierdzenia Cayleya z ciągami Prüfera

Dla n∈{1,2}n\in \{1,2\}n∈{1,2} jest OK.

Mamy drzewo TTT posiadające nnn wierzchołków oraz (narazie) pusty ciąg PPP.

Niech b1b_1b1​ będzie najmniejszą liczbą będącą label-em liścia drzewa TTT, za to a1a_1a1​ będzie liczbą będącą label-em sąsiada tego wierzchołka z label-em b1b_1b1​.

Usuwamy wierzchołek z label-em b1b_1b1​ i krawędź łączącą go z resztą TTT i do ciągu PPP append-ujemy a1a_1a1​.

Powtarzamy proces, aż nie otrzymamy P=(a1,a2,…,an−2)P = (a_1,a_2,\dots,a_{n-2})P=(a1​,a2​,…,an−2​) a drzewo TTT będzie K2K_2K2​ (będzie miało tylko dwa wierzchołki).

Algorytm wyznaczania ekscentryczności wierzchołka

Zakładamy, że dla każdego wierzchołka xxx mamy metodę wyznaczania listy jego sąsiadów N(x)N(x)N(x).

INPUT : v∈Vv \in Vv∈V
OUTPUT : ecc(v)\mathrm{ecc}(v)ecc(v)

L = 0
visited = [v]
active  = N(v)
while active != []:
  L += 1 # (1)
  visited += active
  active = [y for x in active for y in N(x) if y not in visited]

return L
