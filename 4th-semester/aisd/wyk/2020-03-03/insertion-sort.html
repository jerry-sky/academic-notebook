Insertion sort(A,n)

A={a1,a2,...an}A = \{ a_1, a_2, ... a_n \}A={a1​,a2​,...an​}

for j = 1 to n {
  key = A[j]
  i = j-1
  while( i >= 0 && A[i] > key) {
    A[i+1] = A[i]
    i--
  }
  A[i+1] = key
}
