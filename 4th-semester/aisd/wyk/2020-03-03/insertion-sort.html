Insertion sort(A,n)

\(A = \{ a_1, a_2, ... a_n \}\)

for j = 1 to n {
  key = A[j]
  i = j-1
  while( i >= 0 && A[i] > key) {
    A[i+1] = A[i]
    i--
  }
  A[i+1] = key
}
