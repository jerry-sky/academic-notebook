Description

Select jest pięknym przykładem nietrywialnego wykorzystania metodologii dziel i zwyciężaj.

Składa się on z 5 kroków (w tym dwa rekurencyjne).

Algorithm

  1. Najpierw znajdowany jest element xxx tablicy AAA zwany medianą median, dla którego zapewnione jest, że liczba elementów tablicy AAA większych od xxx jest nie mniejsza niż 3n10−6\frac{3n}{10} - 6103n​−6.
  2. Następnie wykorzystuje się element xxx jako pivot do procedury partition i wykonuje dalsze dalsze kroki podobnie jak to miało miejsca w przypadku algorytmu RandomSelect.

Complexity
